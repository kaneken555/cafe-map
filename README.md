## 🛠 技術スタック
- **フロントエンド:** React (TypeScript)
- **バックエンド:** Django (Python)
- **データベース:** PostgreSQL
- **クラウド:** AWS
- **コンテナ:** Docker
- **API:** Google Maps API


## 🚀 セットアップ手順

### 1. **リポジトリのクローン**
```bash
git clone https://github.com/kaneken555/cafe-map.git
cd cafe-map
```

### 2. **Dockerコンテナの起動**
```bash
docker-compose up --build -d
```
